# Boss Weapons Mod
his is my first mod, so it's probably not very good. It adds some weapons for each mob that is generally considered a boss (Wither, Ender Dragon, Warden, and Elder Guardian). 
This mod is free and Open-Source, and I permit forks as long as credit is given to the original mod. This mod was made using MCreator, and was ready a while ago, but I wanted to give it some time to make sure no massive bugs were found. I am open to feedback, but please give criticism in a helpful and constructive way if possible.
This mod is available exclusively on Modrinth and Github, if you installed it anywhere else, it is not an authorised release and may contain malware, only download from these sources unless you are sure that the external source is a safe fork of this mod.
